## 一键部署 (GitHub Pages)

1. 访问 https://github.com/new 创建新仓库
2. 仓库名: `toolbox` (或任意名称)
3. 将 `toolbox/` 文件夹内的 **所有内容** 上传到仓库
4. 进入仓库 Settings > Pages
5. 选择 main 分支，根目录 `/`
6. 等2分钟，你的网站在 `https://你的用户名.github.io/toolbox/` 上线
